# Smart Pothole Detection System

## Overview
This project detects potholes using YOLOv8 and provides a dashboard for monitoring and analytics.

## Features
- Pothole detection using deep learning
- Severity classification (Minor, Moderate, Severe)
- Report generation and storage (JSON)
- Dashboard with analytics
- Map visualization

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Run the app:
streamlit run phase2/dashboard/app.py

## Note
Model weights (.pt) and datasets are not included due to size.
