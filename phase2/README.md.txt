# Phase 2 – Smart Pothole Detection System

This phase extends the pothole detection system developed in Phase 1 by introducing
a dashboard for visualization, report tracking, and basic analytics.

Modules Implemented:
- Module 3: Backend Dashboard
- Module 4: User History & Feedback
- Module 5: Basic Analytics (Visualization)

Technology Stack:
- Streamlit
- JSON backend
- Folium (maps)
- Matplotlib / Seaborn
